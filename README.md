# Lab37 Robotics Product Manager Candidate Campaign

Independent candidate materials by Russell Dudek for the Lab37 Robotics Product Manager opportunity.

## Strategic argument

Bowl Builder is a complete product system, not an isolated robot. The campaign demonstrates a product-management approach that connects recipe behavior, electromechanical performance, order truth, operator flow, serviceability, and economics through one integrated product contract.

## Routes

- `index.html` — candidate vision and interactive Recipe-to-Release Assembly
- `resume.html` — exactly two printable pages
- `cover-letter.html` — exactly one printable page
- `interview-brief.html` — four-page interview thesis brief
- `90-day-plan.html` — three-page entry plan
